# Sintaxe e Operadores

## Tipos de Operadores

### Aritmética

**Operador: Descrição**

- +: Adição
- -: Subtração
- \*: Multiplicação
- \*\*: Expotenciação
- /: Divisão
- %: Módulo (Resto da Divisão)
- ++: Incrementar
- --: Decrementar

### Atribuição

**Operador --> Exemplo ---> Equivalente a**

- **=** --> x = y --> x = y
- **+=** --> x += y --> x = x = y
- **-=** --> X -= y--> X = x - y
- **\_=** --> x \_= y --> x = x \* y
- **/=** --> x /= y --> x = x / y
- **%=** --> x %= y --> x = x % y

### Operador de Comparação

**Operador: Descrição**

- == Igual a
- === mesmo valor e mesmo tipo
- != Diferente
- !== Valor e tipos diferentes
- \> maior que
- \> menor que
- \> = Maior ou igual
- <= Menor ou igual>

### Operador Lógico

**Operador: Descrição**

- && "e" Lógico Ambas as informações devem ser verdadeiras
- || "ou" Qualquer informação precisa ser verdadeira
- ! "Não" Acessar o dado oposto, inverte

### Condicional

(Condição) ? (expressão 1- Verdadeira) : (expressão 2 - Falso)
